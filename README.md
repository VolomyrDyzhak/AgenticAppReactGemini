**ChefOS** es un generador de recetas inteligente que utiliza el motor agéntico de **Gemini 3.1 Flash** para transformar ingredientes cotidianos en menús de alta cocina. Diseñado con una interfaz inspirada en los sistemas operativos modernos y una estética *dark mode* minimalista.



## 🚀 Características

- **Motor Agéntico:** Utiliza modelos de última generación (Gemini 2.0/3.1) para razonar sobre la combinación de ingredientes.
- **Interfaz Glassmorphism:** Diseño ultra moderno con desenfoques en tiempo real y gradientes dinámicos.
- **Validación de Repostería:** Capaz de generar recetas técnicas complejas (como Macarons) detectando proporciones e ingredientes clave.
- **Arquitectura de Estados:** Gestión fluida de vistas (Picker -> Results -> Detail) para una experiencia de usuario sin fricciones.

## 🛠️ Stack Tecnológico

- **Frontend:** React + TypeScript.
- **Estilos:** CSS3 nativo con Variables Dinámicas y animaciones Keyframes.
- **IA:** Google Generative AI SDK (Gemini API).
- **Herramientas:** Vite para un bundling ultra rápido.
